# Sample Webpage

This workspace contains a simple sample webpage and supporting documentation.

## Files

- `index.html`: A complete, responsive sample webpage with custom styling and animations.
- `README.md`: Instructions and overview.

## How to Run

1. Open `index.html` in any web browser.
2. For live-reload development, open the folder in VS Code and use a local server extension (for example, Live Server).

## What This Page Demonstrates

- Responsive layout for desktop and mobile
- Distinct visual design with gradients and layered backgrounds
- Typography hierarchy and custom buttons
- Basic reveal animations for a polished first impression
